Ejemplo de 'display' y 'visibility'

Sobre este objeto se aplican los cambios a 'display' y a 'visibility'

Activa los enlaces para ver el efecto de los diferentes valores posibles de 'display' y 'visibility'

  • Cambiar 'display' a 'none': El elemento desaparece, y deja de ocupar espacio.
  • Cambiar 'display' a 'block': El elemento se convierte en una caja
  • Cambiar 'display' a 'inline': El elemento se convierte en un elemento en línea: pierde sus propiedades de 'caja'
  • Cambiar 'visibility' a 'hidden': El elemento sigue ocupando el mismo espacio que antes en la página, pero ya no se ve.
  • Cambiar 'visibility' a 'visible': El elemento se muestra.

Valor actual de display : none
Valor actual de visibility : visible
